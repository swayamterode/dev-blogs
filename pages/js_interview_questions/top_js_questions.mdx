# Top JavaScript Interview Questions

This are the output based questions that are frequently asked in JavaScript interviews.

## Scope of Variables in JavaScript

What will be the output of the following code?

#### Question 1

<details>
<summary> Click to see the code </summary>

```js filename="output-based-questions.js" showLineNumbers
{
  let a = 1;
  let b = 2;

  console.log(a);
  console.log(b);
}
console.log(a);
console.log(b);
```

Output:

<details>
<summary> Click to see the output </summary>

The following output will be displayed:

```bash
1
2
ReferenceError: a is not defined
```

## Hoisting

What will be the output of the following code?

#### Question 1

<details>
  <summary>Click to see the code</summary>

```js filename="output-based-questions.js" showLineNumbers
{
  console.log(a);
  var a = 1;
}
```

Output:

<details>
  <summary>Click to see the output</summary>

The following output will be displayed:

    ```bash
    undefined
    ```

</details>

<Callout type="info">
  The variable `a` is hoisted to the top of the function, so it is accessible
  before it is declared.
</Callout>

</details>

#### Question 2

<details>
  <summary>Click to see the code</summary>

```js filename="output-based-questions.js" showLineNumbers
{
  console.log(a);
  let a = 1;
}
```

Output:

<details>
  <summary>Click to see the output</summary>

The following output will be displayed:

    ```bash
    ReferenceError: Cannot access 'a' before initialization
    ```

<Callout>
    `let` and `const` are not hoisted to the top of the function, so they are not
    accessible before they are declared.
</Callout>
</details>

</details>
